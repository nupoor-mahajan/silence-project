System Workflow

Step-by-step pipeline after CSV upload

Working Steps

  1. User uploads complaint dataset in CSV format.
  2. System computes complaint density per district.
  3. Average density baseline is calculated.
  4. Districts below threshold are flagged as “silent zones”.
  5. Outputs include ranked tables + charts + summary metrics.

Outputs Generated

  • Dataset preview table
  • Bar chart + Line trend graph
  • Pie chart distribution
  • Flagged silent district ranking
